git init

git add .

git commit -m "项目初始化"

git remote add origin git@github.com:wanshi-dl/dl_002_demo.git

git branch -M main

git push -u origin main
第一次提交

git push
往后提交

git checkout -b login
创建指定名称的新分支 ，并 立即切换到新分支上

git branch
查看当前Git 仓库中所有的分支列表


git push -u origin login
分支提交

git checkout master
切换到master分支

git merge login
在master分支上运行git merge命令，将login分支的代码合并到master分支


git clone 远程仓库地址
